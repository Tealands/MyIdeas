
# 新世代結婚相談所サービス 企画提案書

**コンセプト： 「スペック」で出会い、「生活解像度」で結ばれる**

作成日:2026/4/12

## 1. 背景と目的

従来の結婚相談所は年収や学歴といった「静的なステータス」の比較に終始しており、成婚後の「生活の不一致」による早期離婚や、お見合い時の緊張による「本来の人間性の埋没」が課題となっています。
本提案では、**「労働環境（福利厚生）」と「実生活での振る舞い」を可視化**することで、真に持続可能なマッチングを実現することを目的とします。

---

## 2. 提案の柱（コア・バリュー）

### ① 「生活の質」を可視化するアドバンスド・プロフィール

単なる額面年収だけでなく、結婚後の可処分所得や時間を予測できる項目を追加します。

* **福利厚生のタグ化:** 年間休日数、有休取得率、家賃補助、家族手当、育休取得実績、リモートワーク可否など。
* **家計・家事スタンスの明示:** 資産運用（iDeCo/NISA）への理解、家事分担の希望割合、外食・自炊の頻度など。

### ② 「行動観察型」マッチング：集団BBQ制度

1対1の面談では取り繕えてしまう人間性を、集団の中での「役割」から見極めます。

* **実施内容:** 月1回程度の定期開催。あえて役割を固定せず、準備・調理・片付けのプロセスを共有。
* **評価ポイント:** * 困っている人への声掛け（共感性）
  * 段取りの良さ（問題解決能力）
  * 食べ方やマナー（生活習慣）

### ③ 専門家による「振る舞いフィードバック」

集団イベント中の行動をカウンセラーが分析し、本人にレポート。

* **自己改善の促進:** 「無意識の癖」を修正し、成婚率を向上。
* **信頼性の担保:** 相手に対し、スタッフが保証する「第三者から見た長所」を提示可能に。

---

## 3. 具体的なビジネスフロー

| **フェーズ**       | **内容**                                                  |
| ------------------------ | --------------------------------------------------------------- |
| **入会・登録**     | 福利厚生を含む「生活データ」を詳細にヒアリング。                |
| **マッチング**     | AIが「休日の一致」や「家事価値観」を優先してレコメンド。        |
| **集団イベント**   | **BBQ制度**へ参加。複数の異性と同時に「共同作業」を行う。 |
| **振り返り**       | スタッフが行動をフィードバック。相性の良かった相手を特定。      |
| **お見合い・交際** | 共通の「生活解像度」を持った状態で、深い会話に集中。            |

---

## 4. 競合他社との差別化ポイント

| **項目**           | **従来の相談所**     | **本提案のモデル**                     |
| ------------------------ | -------------------------- | -------------------------------------------- |
| **判断基準**       | 額面年収・外見・学歴       | **可処分時間・家計管理・行動特性**     |
| **出会いの場**     | ホテルのラウンジ（非日常） | **BBQ・共同作業（日常の延長）**        |
| **ミスマッチ防止** | 交際中の手探り             | **入会・イベント時点での価値観一致**   |
| **ターゲット層**   | 条件重視の層               | **共働き・ワークライフバランス重視層** |

---

## 5. 期待される効果

1. **成婚率の向上:** お見合いの「会話」だけでなく、BBQでの「行動」が加点要素になるため、口下手な実務派（エンジニア等）の成婚率が高まります。
2. **離婚率の低下（LTVへの寄与）:** 休日の過ごし方や金銭感覚、家事能力を納得した上で成婚するため、結婚後のトラブルを最小化できます。
3. **ブランド力強化:** 「最も結婚後のリアルを考えている相談所」としてのポジションを確立し、20代〜30代の共働き志向層を独占します。
