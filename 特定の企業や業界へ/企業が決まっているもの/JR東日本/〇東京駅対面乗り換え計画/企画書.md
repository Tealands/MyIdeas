# 企画案：東京駅新幹線プラットフォーム再編計画

**～19番線の東北新幹線転用による「日本縦断対面乗り換え」の実現～**

作成日:2026/4/12

## 1. 背景と目的

現在、東京駅の新幹線ホームはJR東海（東海道）とJR東日本（東北・上越・北陸）で完全に分離されており、乗り換えには改札の通過と階層移動を伴う。

リニア中央新幹線の開業により東海道新幹線の輸送力に余裕が生まれることを好機と捉え、歴史的に東北新幹線用として確保されていた19番線を本来の用途に戻し、広域鉄道ネットワークの利便性を劇的に向上させる。

## 2. 現状の課題と前提条件

* **東海道側の飽和:** 現状、14〜19番線（3面6線）をフル活用しても「のぞみ12本ダイヤ」が限界に近い。
* **東北側の制約:** 20〜23番線（2面4線）のみで5方面の新幹線を捌いており、ダイヤの柔軟性が欠如している。
* **歴史的経緯:** 18・19番線ホームはJR東日本の所有資産をJR東海が借用している形態であり、返還に関する潜在的な合意形成の余地がある。

## 3. 実施計画：リニア名古屋開業後の再編

### ① 東海道新幹線ダイヤの再編

* 「のぞみ」の主要役割をリニアへ移行し、東海道新幹線を「ひかり」「こだま」中心のダイヤへシフト。
* 使用ホームを14〜18番線（3面5線）に集約し、19番線をJR東日本へ返還・移管。18番線は引き続きJR東海が使用する。

### ② 19番線の物理的・技術的転用

* **配線改良:** 19番線北側の引き上げ線を撤去し、秋葉原方面の東北新幹線本線へ接続。
* **電気設備:** 19番線の架線周波数を60Hzから50Hzへ変更。18・19番線間に電気的境界を設置。
* **運行管理:** 19番線の信号システムをJR東日本の「COSMOS」へ統合。

## 4. 期待される効果

### ユーザー利便性の最大化（対面乗り換えの実現）

* **18番線（東海道終着）と19番線（東北始発）による同一ホーム乗り換え。**
* 名古屋・大阪方面から仙台・金沢・札幌（将来）方面への移動における心理的・物理的障壁を排除。

東海道新幹線はこだま、東北新幹線はやまびこ・あさま・たにがわ等の停車駅が多い列車を中心に配置することで航空機と需要をかぶせないようにする。これにより、両者の利用者が相互に乗り換えやすくなり、東京駅通過する広域ネットワークの利便性が大幅に向上する。

### 運行オペレーションの効率化

* JR東日本側が5線化されることで、折り返し時間に余裕が生まれ、雪害等によるダイヤ乱れ時の回復力が向上する。
* JR東海側も、リニアとの役割分担により、東京駅での過密な清掃・折り返し作業の負担が軽減される。

## 5. 課題と対策

| **課題**           | **対策**                                                                                 |
| ------------------------ | ---------------------------------------------------------------------------------------------- |
| **会社間清算**     | 所有権に基づき、賃貸借契約を円満に解消。駅構内店舗（エキュート等）の収益分配を再設計。         |
| **工事中の運用**   | 東海道側の本数が減るリニア開業後の閑散期を利用し、段階的に配線切り替えを実施。                 |
| **境界改札の扱い** | 両改札を統合するか、18・19番線ホームを「共通自由ゾーン」として定義(その場合別々の出口を作る)。 |

---

## 6. 結言

本計画は、国鉄分割民営化から続く東京駅の構造的制約を解消し、日本の二大新幹線網を「点」ではなく「線」で結ぶものである。リニア開業という100年に一度の機会を、単なる新線建設に留めず、既存インフラの最適化・高度化に繋げるべきである。
