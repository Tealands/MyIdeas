# 企画提案書：上野駅新幹線連絡階「恒温モバイル・ワークベース」設置計画

**提案者：** 〇〇大学 工学部 情報工学科 2年 [あなたの氏名]

**コンセプト：** 地下の安定環境と「1分10円」の即時決済が融合する、次世代駅ナカワークスペース

---

## 1. 背景と目的

上野駅新幹線ホーム（地下4階）および連絡階（地下3階）は、年間を通じて気温が安定している（夏涼しく冬暖かい）という物理的利点があります。この未利用空間を、モバイルユニット形式のコワーキングスペースとして活用することで、以下の目的を達成します。

* **出張者の利便性向上:** スーツケースを持つ旅行者の「乗車前の隙間時間」の有効活用。
* **新たな駅収入の創出:** 入場券収入とスペース利用料のダブル収益。
* **低コスト・低リスク運用:** 建築工事を伴わないモバイル形式による試験導入。

## 2. サービス概要

* **名称:** UENO Deep-Work Station（仮称）
* **場所:** 上野駅 新幹線改札内 地下3階 連絡階コンコース
* **規模:** 30席（モバイルユニット形式）
* **料金体系:** **1分 10円（税込）**
  * 交通系ICカード（Suica等）による入退室時の自動秒単位決済。
  * 入場券（150円/2時間）のルールを準用。

## 3. 設備・技術仕様（工学的アプローチ）

「物を提供せず、環境とインフラを提供する」ことに特化します。

| **項目**     | **具体的な仕様**                 | **理由・メリット**                                                       |
| ------------------ | -------------------------------------- | ------------------------------------------------------------------------------ |
| **什器**     | キャスター付き連結デスク（幅120cm〜）  | **スーツケースを足元に置ける**広さを確保。配置変更や他駅への転用が容易。 |
| **椅子**     | 人間工学に基づいた高機能チェア         | 「1分10円」以上の付加価値（疲労軽減）を実感させ、リピート率を向上。            |
| **通信**     | Wi-Fi 6E対応 高密度アクセスポイント    | 地下特有の電波干渉を抑え、Web会議に耐えうる安定した帯域を確保。                |
| **配線**     | パワーポール方式（床工事なし）         | 天井や既存の柱から電源を供給。駅施設の原状回復を容易にする。                   |
| **空席管理** | 赤外線距離センサによるリアルタイム検知 | 1階改札付近のサイネージに空席状況を表示し、地下への誘導を促す。                |

## 4. ビジネスモデルの優位性

1. **高回転・高納得感:** 1分単位の課金により、新幹線の発車時刻ギリギリまで利用したいユーザーの心理に合致。
2. **オペレーションレス:** 無人決済・スマートロック・リモート監視により、人件費を極限まで削減。
3. **環境負荷の低減:** 地下の恒温性を利用することで、地上施設に比べ空調エネルギー消費を約20%〜30%抑制可能（試算）。

## 5. ロードマップ（試験導入案）

* **フェーズ1（準備）:** 連絡階の電波強度調査（サイトサーベイ）および騒音測定。
* **フェーズ2（試行）:** 1ヶ月間の限定設置。利用者の属性データ（滞在時間、利用時間帯）を収集。
* **フェーズ3（展開）:** 利用データに基づき、大宮・仙台・盛岡等の新幹線停車駅へユニットを転用・拡大。
