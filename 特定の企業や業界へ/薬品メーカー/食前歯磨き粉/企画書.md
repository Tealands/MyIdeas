# 新商品開発企画書：食前専用ハミガキ「Gourmet Prep（グルメ・プレップ）」

## 1. コンセプト背景

現代の衛生意識の高まりにより、「帰宅直後」や「起床直後」の食事前に歯を磨く習慣が定着しつつあります。しかし、従来の歯磨き粉に含まれる界面活性剤や強いミント香料は、舌の味細胞を一時的に麻痺させ、食味を大きく損なう課題（オレンジジュース効果）がありました。

本プロジェクトは、**「食事をより美味しく楽しむための、味覚をリセットする食前習慣」**を提案します。

## 2. ターゲットユーザー

* **美食家・ワイン愛好家:** 食材本来の繊細な風味を大切にする層。
* **健康志向のビジネスパーソン:** 帰宅後のウイルス対策と食事をシームレスに行いたい層。
* **QOL（生活の質）重視層:** 日常の食事を「作業」ではなく「体験」として向上させたい層。

## 3. 製品の3大特徴（コア・バリュー）

1. **味覚修飾の完全排除 (Taste-Neutral Formula)**
   * ラウリル硫酸ナトリウム（SLS）を一切排除し、味蕾に影響を与えない天然由来の洗浄成分を採用。
2. **アフターケアから「プレ・ケア」へ**
   * フッ素の代わりにハイドロキシアパタイトを使用。歯面を整え、食後の着色汚れ（ステイン）を付きにくくするガード機能を付与。
3. **クイック・リンス設計**
   * 食事を待たせないよう、粘度を調整。1回のすすぎで成分が口内に残らないキレの良さを実現。

## 4. 推奨成分構成（案）

| **分類**   | **推奨成分**       | **採用理由**                             |
| ---------------- | ------------------------ | ---------------------------------------------- |
| **清掃剤** | ヒドロキシアパタイト     | 歯の平滑化、着色予防。無味無臭。               |
| **洗浄剤** | キラヤエキス（サポニン） | 天然由来。味覚への影響を最小限に抑えた微発泡。 |
| **湿潤剤** | エリスリトール           | 口内の菌を分散し、スッキリとした後味を演出。   |
| **香味剤** | 超微量レモンエキス       | 唾液分泌を促し、味覚を活性化（食欲増進効果）。 |

## 5. 製造パートナー候補

* **日本ゼトック株式会社:** 高機能な薬用処方と、特殊な使用感の調整に強み。
* **太陽油脂株式会社:** 合成界面活性剤フリーの製造ラインと、ナチュラル製品のブランド力。

## 6. 市場展開イメージ

* **販売チャネル:** セレクトショップ、高級スーパー、ワインショップ（レジ横）、歯科医院。
* **パッケージ:** 既存の「衛生用品」然としたデザインではなく、キッチンや食卓に置いても違和感のない、調味料や化粧品のような洗練されたデザイン。
