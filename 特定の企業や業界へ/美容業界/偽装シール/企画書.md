
# 新商品開発企画書：Stealth Patch（ステルス・パッチ）

作成日:2026/4/11

## 1. コンセプト

**「完璧な素顔を、戦略的なノイズで守る」**

従来の「欠点を隠す」メイクアップとは真逆の、**「偽の欠点（ノイズ）を付与することで、真実の造形をプロテクトする」**という逆転発想のプライバシー保護アイテム。

## 2. ターゲット層

* **著名人・インフルエンサー:** プライベートで「オーラ」を消し、平穏に街を歩きたい層。
* **プライバシー意識の高い一般層:** SNSへの無断映り込みや、AI顔認証による特定を避けたい層。
* **ストーカー・防犯対策:** 帰宅時などに「特定の顔立ち」という記憶を攪乱したい人。
* **Z世代・クリエイター:** 「あえて不完全な自分」を演出する新しいファッション表現。

## 3. 製品ラインナップ（3つの視覚ノイズ）

| **カテゴリ**   | **商品名（仮）**                            | **視覚効果・メカニズム**                                                               |
| -------------------- | ------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| **ニキビ**     | **Acute Point (アキュート・ポイント)**      | 3D構造のハイドロコロイド。視線を一点に強烈に誘導し、目や鼻の造形から意識を逸らす。           |
| **そばかす**   | **Texture Blur (テクスチャ・ブラー)**       | 広範囲に点在する色素ドット。肌の質感を「野生化」させ、都会的で洗練された印象を中和する。     |
| **シミ・アザ** | **Asymmetry Mask (アシンメトリー・マスク)** | 左右非対称な色素沈着を再現。人間の脳が好む「顔の対称性（美しさ）」を破壊し、記憶を攪乱する。 |

## 4. 技術的特徴（情報工学的アプローチ）

* **特徴点攪乱（Feature Point Noise）:** AIの顔認証アルゴリズムが抽出する「目尻」「小鼻」「口角」付近に配置することで、マッチング精度を意図的に低下させる。
* **マット・フィニッシュ技術:** 特殊な低反射素材を採用。カメラのフラッシュや強い照明下でも「テカリ」が出ず、本物の皮膚トラブルに見える質感を追求。
* **ワンタッチ・ディスカード:** 医療用低刺激粘着剤を使用。貼る時は1秒、剥がす時は洗顔で容易に除去可能。

## 5. 販売戦略（Amazon・D2C特化）

* **プロモーション:** * 「芸能人のステルス術」という文脈でインフルエンサーによる検証動画を公開。
  * 実際に「どれだけ気づかれないか」の実験データを掲載し、情報の真実性を担保する。
* **パッケージング:** * 薬局のような「治療薬」風のデザインではなく、ガジェットやサイバーセキュリティー用品のようなクールなデザインを採用。
* **セット販売:** * 「ステルス・フルセット（ニキビ＋そばかす＋シミ）」に、レンズをあえて曇らせた「ダミー眼鏡」を同梱。

## 6. 市場展開のフェーズ

1. **Phase 1:** Amazonおよび自社ECサイトでの限定販売。ニッチな「変装・防犯」需要を固める。
2. **Phase 2:** コスプレ・舞台用メイクアップ市場への拡大。
3. **Phase 3:** 「SNS映え」の反動としてくる「アンチ・パーフェクト（不完全美）」トレンドへの乗船。
