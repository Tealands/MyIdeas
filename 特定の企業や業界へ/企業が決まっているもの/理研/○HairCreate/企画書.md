# HairCreate 企画提案書

## 1. 企画概要

**プロフェッショナル向けヘアスタイル設計・シミュレーションソフトウェア「HairCreate」**の共同研究・開発に関する提案である。

HairCreate は、機械設計分野における CAD/CAE の思想をヘアデザイン分野へ拡張し、
これまで職人技・経験知に依存してきたヘアスタイル創作を
**数理モデル・AI・シミュレーションにより再定義すること**を目的とする。

本企画では、

* 既に高度なシミュレーション技術・AI技術・運用実績を有する研究機関・開発企業
* 長期的な研究開発および社会実装を視野に入れた運営体制

を前提とし、共同でのソフトウェア開発を想定する。

---

## 2. 背景と課題認識

### 2.1 ヘアデザイン分野の現状

* ヘアスタイル設計は高度に構造的であるにもかかわらず、
  設計プロセスが形式知化されていない
* 新スタイル開発は属人的・試行錯誤的
* 企業内・国際拠点間での設計共有が困難
* 再現性・量産性の評価指標が存在しない

### 2.2 既存ツールの限界

| 分類     | 主用途     | 限界       |
| ------ | ------- | -------- |
| 試着アプリ  | 顧客向け可視化 | 創作・設計不可  |
| 画像生成AI | ビジュアル生成 | 再現不能     |
| CGツール  | 映像制作    | ヘア設計概念なし |

**「創るための設計ツール」そのものが存在しない**ことが最大の課題である。

---

## 3. HairCreate のコンセプト

### 3.1 定義

> HairCreate は、ヘアスタイルを工学的に設計・検証・共有するための
> プロフェッショナル向けヘアデザイン CAD/CAE ソフトウェアである。

### 3.2 CAD/CAE との対応関係

| 機械設計  | HairCreate          |
| ----- | ------------------- |
| 部品    | 毛束（Strand / Bundle） |
| アセンブリ | ヘアスタイル全体            |
| 寸法    | 髪の長さ・角度・分布          |
| 材質    | 髪質（太さ・硬さ・摩擦・光の反射）        |
| 荷重    | 重力・風・動作             |
| 解析    | 崩れ・ボリューム変化          |
| 図面    | カット設計データ            |

---

## 4. システム機能概要

### 4.1 設計機能（CAD）

* 頭部3Dモデル（スキャンまたはパラメトリック）
* 毛束単位での設計・編集
* レイヤー構造・分け目・毛流れ方向の定義
* 長さ・密度・体積分布の数値指定

### 4.2 シミュレーション機能（CAE）

* 重力・動作・風による形状変化
* 濡れ／乾き状態の差異
* 髪質差によるボリューム変化
* 再現性・安定性の評価

### 4.3 AI 支援機能

AI は自動生成ではなく、**設計補助エンジン**として機能する。

* 初期設計案の提示
* 写真・3Dスキャンからのパラメータ推定
* 設計変更時の影響予測
* 過去設計データからの類似検索

## 5. 想定ユーザー・用途

* 化粧品・ヘアケアメーカー（新ライン向けスタイル設計）
* ウィッグ・医療用ヘア製品メーカー
* コレクション・広告向けヘアデザイナー
* 教育機関（設計と実技の橋渡し）
* CG・映像制作（実髪と整合した設計）

---

## 6. 開発体制の想定

本企画は以下のような組織が主導することを想定する。

* 高度な数理・物理シミュレーション技術
* AI・機械学習研究の実績
* 長期運営・社会実装経験

（例：理化学研究所発ベンチャー、研究機関系ソフトウェア企業 等）

提案者は、

* ヘアデザイン分野の構造整理
* 要件定義・概念設計
* 業界側知見の提供

を担い、共同研究・共同開発の形を想定する。

## 7. 開発ロードマップ（案）

* 企業導入
* 標準データ形式化

## 9. まとめ

HairCreate は単なる美容ソフトではなく、
**ヘアデザインを工業設計の次元へ引き上げる基盤技術**である。

本企画は、

* 研究機関・高度技術企業にとっては新たな応用分野
* ヘア業界にとっては構造的変革

をもたらす可能性を持つ。
