# 📄 情報発信責任法（IPL法）制定企画書（最終骨子案）

---

## 1. 企画概要：情報社会の信頼性確保と法的位置づけ

| **項目**           | **内容**                                                                                                                                                                                         |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **法案名称（案）** | **情報発信責任法（IPL法：Information Provider Liability Law）**                                                                                                                                  |
| **法的目的**       | 情報の発信者に対し、その情報に**重大な誤りまたは虚偽**があった場合に、**刑事罰または民事賠償責任**を負わせることで、情報流通における**信頼性**を確保し、国民の権利利益の保護を図る。 |
| **法の性質**       | 製造物責任法（PL法）の考え方を応用した、**情報流通における特別法** 。                                                                                                                            |
| **適用分野**       | 公衆に経済的損害、健康被害、安全への脅威をもたらす可能性のある**客観的な事実情報** 。                                                                                                            |

---

### 2. 適用対象者と「欠陥情報」の定義

#### 2.1 適用対象となる情報発信者

情報の**内容を実質的に決定し、広範な不特定多数に向けて発信する主体**を指します。

* **📰 事業体:** ニュースメディア、出版社、放送局、経済的利得を伴う情報提供サービス運営者。
* **👤 特定の個人:** 専門家としての肩書を持ち、継続的かつ影響力の大きい情報発信を行う者（例：登録者数や閲覧数が一定以上のインフルエンサー）。
* **適用除外（検討）:** 私的な会話、個人的な日記、学術研究目的の公表、政治的主張、純粋な意見・論評。

#### 2.2 責任が問われる「欠陥情報」の定義

責任を問う対象は、社会的な影響や損害の大きさを考慮した「欠陥情報」に限定されます。

* **客観的な誤認または虚偽:** 一般的に正しいと認識されるべき**客観的な事実**について、誤認を招くまたは虚偽の内容であること。
* **社会的な影響:** 経済的な損害、健康被害、公衆の安全の脅威など、**他者に重大な不利益**をもたらす可能性のある情報であること。

---

### 3. 責任の種類と要件

#### 3.1 民事責任（損害賠償）

過失責任を原則としつつ、影響力の大きい事業者には厳格な責任を課します。

| **責任の原則**         | **要件**                                                                                                                               | **特記事項**                                |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| **過失責任（基本）**   | 1. 欠陥情報の発信。 2. 発信者に**過失** （注意義務違反）。 3. 欠陥情報と損害の **因果関係** 。                                   | 財産的損害および精神的損害を賠償。                |
| **無過失責任（特例）** | 経済的利得を伴う情報提供事業を行う法人は、情報が欠陥であったことのみで責任を負う。ただし、**過失の欠如**を発信者側が立証できれば免責。 | **立証責任の転換** （PL法と同様の考え方）。 |

#### 3.2 刑事責任（懲役・罰金）

**意図的**または**極めて重大な過失**による、公衆への看過しがたい損害・危険に限定します。

| **類型**       | **要件**                                                                                                                           | **法定刑（案）** |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| **故意犯**     | 意図的な**虚偽**の発信により、公衆の生命、身体、または大規模な経済秩序に**実害**を与えた場合。                               | 懲役または高額罰金     |
| **重大過失犯** | 専門家として**極めて高い注意義務**を著しく怠り、公衆の生命、身体、または公共の安全に**現実的な危険・損害**を発生させた場合。 | 罰金または懲役         |

**【刑事責任の具体的な適用例】**

| **事例**       | **発信者の行為と結果**                                                                                             | **責任の根拠**                                               |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------ |
| **経済的混乱** | 株価操作を目的に、**虚偽の企業倒産情報**を意図的に発信し、多数の投資家に甚大な経済的損失を負わせた。               | **意図的な虚偽**による経済秩序の毀損。                       |
| **健康被害**   | 専門家が、**科学的根拠のない誤った治療法**を推奨し、正規治療を中止した患者に**健康被害や死亡**を発生させた。 | **極めて重大な過失**による公衆の生命・健康への危険。         |
| **公共の安全** | 報道機関が災害時、**古いデータに基づき**誤った緊急避難情報を長時間放送し、住民の避難行動中に負傷者を発生させた。   | 災害時における**重大な注意義務違反**による公衆の安全の脅威。 |

---

### 4. 発信者側の抗弁（免責事由）

発信者が以下の事実を立証した場合は、責任を負わないものとします。

1. **相当な注意義務の履行:** 情報の正確性を確保するために、 **一般的に必要とされる最高水準の注意** （取材、査読、根拠資料の確認など）を尽くしていたこと。
2. **訂正措置の迅速な実施:** 欠陥が判明した後、**直ちに適切な方法で訂正または取り消しの措置**を講じたこと。
3. **技術的に予見不可能:** 発信時において、科学的・技術的な知見からその情報が欠陥であることを**予見することが不可能**であったこと。

---

### 5. 課題と今後の検討事項

本法案は、**表現の自由（憲法第21条）**との調和が最大の課題となります。

* **表現の自由との調和:** 適用範囲を厳格に限定し、純粋な意見や論評にまで責任を及ぼさないよう、**「客観的な事実情報」**にのみ適用することを徹底する必要があります。
* **真実性の立証:** 裁判所が客観的に「欠陥」を判断できる、中立的な **判断基準** （例：公的機関の発表、確立された科学的知見）を設定する。
* **国際的な対応:** 国境を越えた情報発信について、どの国の法律を適用するか（国際私法）の明確化。
