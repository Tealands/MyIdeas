# コスト試算表：ビジネス・アピアランス・サポート（BAS）導入モデル

**想定規模：1店舗あたり（営業担当 15〜20名程度）**

---

## 1. 初期導入コスト（イニシャルコスト）

導入初年度にかかる設備投資費用です。

| **項目**                 | **概算費用**  | **備考**                                 |
| ------------------------------ | ------------------- | ---------------------------------------------- |
| **可動式理美容椅子**     | 150,000円           | 横になれる仕様、中古・簡易型なら安価に抑制可能 |
| **照明・鏡設備**         | 30,000円            | プロの施術に必要な高演色LEDライト、姿見        |
| **備品（スチーマー等）** | 20,000円            | 蒸しタオル用スチーマー、洗浄設備代替           |
| **法務・保健所手続き**   | 10,000円            | 出張理容・美容に関する届出手数料               |
| **合計**                 | **210,000円** |                                                |

---

## 2. 月間運用コスト（ランニングコスト）

週5日（月20日）、毎朝2時間の稼働を想定した試算です。

### ① 美容師・理容師への報酬（1店舗あたり）

| **項目**                                                | **計算式**     | **月間コスト**     |
| ------------------------------------------------------------- | -------------------- | ------------------------ |
| **固定基本給**                                          | 1日4,000円 × 20日   | 80,000円                 |
| **実績インセンティブ**                                  | 1件700円 × 8件(※1) | 84,000円                 |
| **交通費**                                              | 1日1,000円 × 20日   | 20,000円                 |
| **小計(理容師一人当たり)**                              |                      | **184,000円**(※2) |
| *(※1) 1日8名利用と想定（15分×6名＝90分＋準備・清掃30分）*|             |                     |
| (※2)セールス数÷8人の理容師を雇用する |                   |                         |

### ② 消耗品・その他

| **項目**                   | **備考**                 | **月間コスト** |
| -------------------------------- | ------------------------------ | -------------------- |
| **シェービング剤・化粧品** | プロ仕様の薬剤・スキンケア用品 | 10,000円             |
| **リネン・クリーニング代** | タオル等の洗濯・管理費         | 6,000円              |
| **小計**                   |                                | **16,000円**   |

### **月間運用合計：200,000円**

---

## 3. 投資対効果（ROI）のシミュレーション

本施策を単なる「経費」ではなく「投資」として捉えるための指標です。

* **社員1人あたりのコスト：25,000円 / 月**
  * （月間20万円 ÷ 利用社員8名の場合）
  * 1回あたりのコストは約1,250円。

### **期待される経済効果**

1. **成約率の向上：** * 客単価1,000万円の商材において、第一印象の改善により成約率が**わずか0.2%向上**するだけで、月間のコストは完全に回収可能です。
2. **採用コストの削減：** * ユニークな福利厚生により、人材紹介会社への手数料（1人あたり100万円〜）を1名分浮かせることができれば、半年分の運用費に相当します。
3. **離職防止効果：** * 熟練社員の離職に伴う損失（再採用・教育コスト）を防止することで、中長期的な収益維持に寄与します。

---

## 4. 特記事項

* **キャンセルペナルティ：** 無断キャンセルが発生した場合、インセンティブ分は会社が補填し、当該社員には「次回予約制限」等のペナルティを課すことで美容師のモチベーションを維持します。
* **税務上の扱い：** 「全社員が利用可能な制度（または特定の職種全員が対象）」として整備することで、福利厚生費としての経理処理を目指します。
