# 企画書：公共放送の解体と放送事業の完全民営化・分散化案

## 1. 背景と目的

現在のインターネット普及環境において、特定の組織が独占的に受信料を徴収し、公共放送を維持する合理的根拠が希薄化している。本案は、NHKを解体し、その機能を「市場原理」と「最小限の公的委託」に分散させることで、国民の負担軽減とメディア市場の活性化を図ることを目的とする。

## 2. NHK解体後の機能分担（リソース再配置）

各機能は以下の通り、民間への譲渡または公的アーカイブへと移行する。

| **旧NHK機能**      | **移行先・手法**                     | **財源・運用形態**       |
| ------------------------ | ------------------------------------------ | ------------------------------ |
| **ニュース・報道** | 民間放送局（各社）                         | 広告収益・サブスクリプション   |
| **災害・政見放送** | 民間への委託（ガイドライン制）             | 国からの協力委託費（最小限）   |
| **教育番組**       | 文部科学省特設サイト・YouTube              | 国家予算（教育費）による運営   |
| **ドラマ・アニメ** | 民間制作会社・配信プラットフォーム         | コンテンツ販売・ライセンス収入 |
| **アーカイブ資産** | 公的デジタルライブラリ（国立国会図書館等） | オープンデータ・一部有償利用   |

## 3. 具体的な実施施策

### ① 災害・インフラ報道の「委託制度」への転換

「公共放送」という巨大組織を維持する代わりに、政府が民間放送局（地上波・ネットメディア）と災害時報道に関する契約を結ぶ。

* 災害発生時の優先放送枠の確保。
* 報道ガイドラインの策定と、遵守に対する「災害報道維持費」の支払い。
* 放送設備（電波塔など）の共有インフラ化。

### ② 教育コンテンツの「完全オープン化」

Eテレが保有する膨大な知見や過去の番組資産を、文部科学省の管理下でデジタル化する。

* **YouTube/ニコニコ動画:** 公式チャンネルでの無料公開（広告収益を維持費に充当）。
* **文科省HP:** 検索性の高い学習データベースとしての構築。
* 学校教育現場での自由な2次利用の解禁。

### ③ エンターテインメントの「市場開放」

大河ドラマやアニメなどのIP（知的財産）を民間へ売却、または制作チームを独立させる。

* 人気コンテンツはNetflixや民放、U-NEXT等が引き継ぎ、高品質な制作を継続。
* 受信料に依存しない、グローバルな市場競争力を持つコンテンツへの転換。

## 4. 期待される効果

* **国民負担の消滅:** 年間約7,000億円にのぼる受信料負担がゼロになる。
* **情報の自由競争:** 多様なメディアが切磋琢磨することで、報道の多角化が進む。
* **デジタル化の加速:** 電波リソースの再配分により、次世代通信（6G等）や新しいITサービスへの帯域活用が可能になる。

## 5. 結論

公共放送という「形態」そのものが役割を終えた現代において、機能を解体・再定義することは、情報の民主化と国家コストの最適化に直結する。物理的な組織を解体し、機能を社会全体に分散させることが、次世代のメディア環境における正解である。
