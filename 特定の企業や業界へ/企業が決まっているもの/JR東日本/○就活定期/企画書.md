# 就活定期 企画書

作成日：2026/2/20

## 1. コンセプト

**登録駅からの「自由度の高い」定期券**

就活生が毎回異なる面接先に赴く際の交通費・心理的負担を軽減するため、従来の「特定の目的地が指定された定期券」ではなく、「登録駅（通常は自宅最寄駅）から都内の任意の駅まで乗り放題」という新しい定期券商品を提案する。

---

## 2. 背景・課題

### 就活生が直面する課題

- **交通費の不確実性**: 面接先が毎回異なるため、事前に定期範囲を特定しにくい
- **心理的負担**: 毎回スイカの残高を気にしながら移動する必要がある
- **経済的負担**: 複数の交通機関を利用する場合、費用が嵩む傾向がある
- **利便性の低さ**: 従来定期では柔軟性が不足している

### JR東日本の機会

- 就活需要という時間限定的なニッチ市場
- 新卒採用候補者との接点構築
- 定期外利用者からの既定客化

---

## 3. サービス内容

### 基本スペック

| 項目               | 内容                                           |
| ------------------ | ---------------------------------------------- |
| **対象者**   | 大学生・大学院生・高専生（就活生）             |
| **起点**     | 利用者が登録した駅（通常は自宅最寄駅等）       |
| **到達範囲** | 東京都内の全駅（または事前に定めるエリア内）   |
| **有効期限** | 1ヶ月単位が基本（就活シーズン = 3月～9月中心） |
| **利用資格** | 学生証等で就活生であることを確認               |

### 従来の定期との差別化

| 要素   | 従来定期                     | 就活定期                       |
| ------ | ---------------------------- | ------------------------------ |
| 行き先 | **固定**（〇〇駅まで） | **自由**（都内任意の駅） |
| 柔軟性 | 低い                         | 高い                           |
| 料金   | 安め                         | やや高め                       |
| 対象者 | 全般                         | 就活生限定                     |

### 利用シーン

- 企業説明会・面接への移動
- 複数志望企業を1日で訪問する場合
- 面接後の企業周辺散策
- アクセス不明な場所への移動

---

## 4. ターゲット・市場規模

### 主要ターゲット層

- **大学3年生**（3月～翌年9月）
- **大学院1年生**（4月～翌年9月）
- **高専5年生**（5月～翌年9月）

### 想定市場規模（東京圏）

- 対象学生数: 約30万人/年
- 購買率見込み: 5～10%（実際に赴任回数が多い学生）
- 見込み販売数: 1.5万～3万件/シーズン

---

## 5. 価格設定（案）

### 料金体系の考え方

- **基準**: 従来の通勤定期 vs. 自由度向上分
- **想定**: 月額 ¥8,000～¥12,000程度

### 価格パターン案

#### 案A: シンプル従量制

| 期間 | 料金     | 参考            |
| ---- | -------- | --------------- |
| 30日 | ¥10,000 | 1日あたり ¥333 |
| 60日 | ¥18,000 | 1日あたり ¥300 |
| 90日 | ¥25,000 | 1日あたり ¥278 |

#### 案B: 段階的プラン

- **ライト** (月3～5回利用想定): ¥6,000
- **スタンダード** (月10～20回利用想定): ¥9,500
- **プレミアム** (無制限): ¥12,000

### 競合との価格比較

- 通勤定期（自宅～渋谷）: ¥7,000～8,000 → **就活定期はプレミアム感** で高めの設定も許容か
- ICカード1回200円×50回: ¥10,000 → 就活定期で十分価値

## 6. メリット・利点

### （学生側）

✅ **心理的負担軽減**: 交通費を気にせず移動可能
✅ **利便性向上**: 急な経路変更時の柔軟対応
✅ **経済効率**: 複数社訪問時にコスト削減
✅ **信頼醸成**: スイカ残高不安からの解放

### （JR東日本側）

✅ **新規需要創出**: 就活という時間限定市場
✅ **ブランド向上**: 若年層への好意度向上
✅ **データ取得**: 就活生の移動パターン分析
✅ **将来顧客化**: 就活後も既定客になる可能性
✅ **採用連携**: 就活サイト等との協業による販売チャネル強化

---

## 7. ビジネスモデル

### 収益源

1. **定期券販売** (メイン)

   - 1シーズン（6ヶ月）で 15,000～30,000件想定
   - 平均単価 ¥10,000
   - 売上規模: 1.5～3.0億円/年
2. **デジタル連携** (副)

   - 学生証アプリ化でのデータ協携
   - キャリア企業との関連者紹介

### コスト構造

| コスト項目                   | 概要                                |
| ---------------------------- | ----------------------------------- |
| **販売手数料**         | 大学等の協力機関に販売委託 (5～10%) |
| **ICカード発行**       | 初回のみ数百円                      |
| **システム開発・運用** | 認証・管理システムの構築・保守      |
| **マーケティング**     | 大学のキャリアセンター等での宣伝    |
| **紛失対応**           | カード再発行、サポート対応          |

### 損益分岐点

- 固定費: 約5,000万円/年（システム・営業）と想定
- 変動費率: 約30%（手数料・発行費用等）
- **BEP**: 約8,000～10,000件/シーズン → **十分達成可能**

## 8. 懸念事項・課題

### 運用上の課題

| 課題                             | 対応案                                             |
| -------------------------------- | -------------------------------------------------- |
| **定期券の不正利用・譲渡** | 学生証番号とICカードの紐付け、顔認証オプション検討 |
| **不正乗降の検知**         | AI・機械学習での異常検知（時間帯・経路パターン）   |
| **カード紛失対応**         | デジタル化・スマホアプリ搭載の検討                 |
| **事故時の対応**           | 標準的な保険約款の適用                             |

### ビジネス上の課題

| 課題                   | 対応案                                               |
| ---------------------- | ---------------------------------------------------- |
| **販売促進**     | 大学キャリアセンター・就活サイトとの連携             |
| **競合出現**     | サービスの継続的改善・付加価値化                     |
| **対象層の確保** | 高専・専門学校へのアプローチ拡大                     |
| **季節変動**     | オフシーズンへの応用展開（留学生向け・インターン等） |

## 10. 期待される効果

### 定性的効果

- 就活生のストレス軽減を通じた「いい企業としてのイメージ」構築
- Z世代との関係性深化

### 定量的効果

- **短期** (2027年): 1～2億円程度の新規売上
- **中期** (2029年): 3～5億円規模への成長
- **顧客LTV**: 就活終了後も通勤定期等への転換で ¥50万以上

## 11. 補足：デジタル化の可能性

### スマホアプリ化

- カード不要でスマホのみで利用
- 大学生のデジタル親和性を活かす
- 「JR東日本アプリ」との統合

### データ活用

- 就活生の移動データ → 新路線・駅改修の参考
- キャリア企業への匿名ビッグデータ提供（B2B収益化）

---

## 終わりに

就活定期は、就活生の実際の困りごとに着眼した、**時間限定的で限定的な商品提案** でありながら、JR東日本にとっては新規需要創出かつ若年層ブランディングの好機となる施策です。

**初期投資を抑えつつ、トライアルから本格展開へ段階的に進める** ことで、リスク最小化での事業化が期待できます。
